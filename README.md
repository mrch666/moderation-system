# Система модерации изображений товаров

## Описание
Система для модерации изображений товаров с:
- REST API для приема данных
- Веб-интерфейсом с настройками
- Очередью модерации
- Интеграцией с Telegram

## Архитектура
1. **Backend** (Node.js + Express + PostgreSQL)
2. **Frontend** (React + TypeScript)
3. **Telegram Bot** (Node.js + Telegraf)
4. **База данных** (PostgreSQL + Redis для очередей)

## Основные сущности
- Модерация: изображение, ID товара, статус, модератор
- Настройки: API ключи, серверы, пользователи Telegram
- Пользователи: администраторы, модераторы
