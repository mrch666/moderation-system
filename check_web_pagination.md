# 📋 ПРОВЕРКА ПАГИНАЦИИ В ВЕБ-ИНТЕРФЕЙСЕ

## 🌐 АДРЕС ВЕБ-ИНТЕРФЕЙСА
**http://192.168.1.189:8080**

## ✅ ЧТО ДОЛЖНО БЫТЬ НА СТРАНИЦЕ

### 1. Вкладка "Очередь модерации"
- Перейдите на вкладку "Очередь модерации"
- Должны увидеть список товаров (10 на страницу)

### 2. Элементы пагинации (внизу страницы):
```
┌─────────────────────────────────────────────────┐
│ Страница 1 из 61 (всего: 609 товаров)           │
├─────────────────────────────────────────────────┤
│ ← Назад  1  2  3  4  5  ...  61  Вперед →       │
├─────────────────────────────────────────────────┤
│ Показывать по: [10 ▼]                           │
└─────────────────────────────────────────────────┘
```

### 3. Функциональность:
- **Кнопки страниц**: Клик по номеру страницы → загрузка этой страницы
- **"Назад"/"Вперед"**: Переход между страницами
- **Активная страница**: Выделена цветом (синяя)
- **Выбор количества**: Можно выбрать 5, 10, 20, 50 товаров на страницу
- **Информация**: Показывает текущую страницу и общее количество

### 4. Тестовые действия:
1. **Кликните на страницу 2** → должны загрузиться товары 11-20
2. **Кликните "Вперед →"** → переход на страницу 3
3. **Кликните "← Назад"** → возврат на страницу 2
4. **Выберите "20" в выпадающем списке** → должно стать 20 товаров на страницу
5. **Кликните на последнюю страницу** → переход на страницу 61

## 🧪 ТЕСТ ПРОИЗВОДИТЕЛЬНОСТИ

### Быстрая навигация:
- Переход между страницами должен быть мгновенным
- Не должно быть "зависаний" интерфейса
- Все товары должны отображаться с картинками

### При изменении количества на страницу:
- Автоматический переход на первую страницу
- Корректный пересчет количества страниц
- Сохранение выбора (после обновления страницы)

## 🐛 ВОЗМОЖНЫЕ ПРОБЛЕМЫ И РЕШЕНИЯ

### Если пагинация не отображается:
1. Проверьте консоль браузера (F12 → Console)
2. Проверьте, что API возвращает данные: `http://192.168.1.189:3000/api/moderation/queue?limit=10`
3. Обновите страницу (Ctrl+F5)

### Если кнопки не работают:
1. Проверьте JavaScript ошибки в консоли
2. Убедитесь, что функция `loadQueue(page)` определена
3. Проверьте, что API ключ правильный

### Если не меняется количество товаров:
1. Проверьте функцию `changeItemsPerPage()`
2. Убедитесь, что параметр `itemsPerPage` обновляется
3. Проверьте, что `loadQueue(1)` вызывается с новым лимитом
